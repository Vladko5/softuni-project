Daily Assistent

is a SPA build with Angular. You can easily shedule your time, keep track on your money and save your favorite links.

With the calendar you can shedule your tasks for the week. Just click the Add Task button and set the task data. The required data for task is :

  • - Title

  • - Date

  • - Start hour

  • - End hour

* You can submmit only if all fields are filled.





When you come back from market you can easily put your bill in the Budget section where you can manage your monthly expenses. The required data for this are:

  • - Product name

  • - Price

  • - From where

* You can submmit only if all fields are filled.





If you have favourit links and you dont know where to store it go on Link section. The required data for task is :

  • - Title

  • - Link

  • - Description

* You can submmit only if title and link fields are filled.
